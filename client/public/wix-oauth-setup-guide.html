Quick Start: Register Your App

1 Go to Wix Developer Center

Visit https://dev.wix.com and sign in with your Wix account.

2 Create a New App
  • Click "Create New App"
  • Choose "Website / Business App"
  • Select "Build a client-side app"
  • Give your app a name (e.g., "My College Finance OAuth")
3 Configure OAuth Settings
  • In your app dashboard, navigate to "OAuth" settings
  • Add your redirect URL:
  • Select permissions:

4 Get Your Credentials

After saving, you'll see:

  • App ID (Client ID) - This should match your existing ID or you'll get a new one
  • App Secret (Client Secret) - Copy this securely, you'll need it for token exchange

OAuth Flow Overview

How Wix OAuth Works:
  1. User clicks "Login with Wix"
  2. Redirected to Wix authorization page
  3. User logs in and approves permissions
  4. Wix redirects back with authorization code
  5. Your app exchanges code for access tokens
  6. Use tokens to access member data via Wix APIs

1 Use the OAuth Test Page

Open OAuth Test Page
2 Manual Token Exchange

If you already have an authorization code, you can exchange it using curl:

curl -X POST https://www.wixapis.com/oauth/access \ -H "Content-Type: application/json" \ -d '{ "grant_type": "authorization_code", "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "code": "YOUR_AUTH_CODE" }'

API Endpoints Available

Your app provides these endpoints for OAuth:

  • POST /api/auth/wix-auth-url - Generate authorization URL
  • POST /api/auth/wix-callback - Handle OAuth callback
  • POST /api/auth/wix-exchange-token - Manual token exchange
